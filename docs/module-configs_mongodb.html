(async, inner) connectDB() → {Promise.<boolean>}

Description:

  • Connects to MongoDB using mongoose with configuration from appConfig. Sets up connection event listeners and handles connection errors.

Source:
  • configs/mongodb.js, line 10
Example
// Connect to database
const isConnected = await connectDB();
if (isConnected) {
  console.log('Database connected successfully');
} else {
  console.log('Database connection failed');
}
